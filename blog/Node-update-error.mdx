Node를 18버전에서 20버전으로 업데이트를 하고싶었습니다. 그래서

```terminal
npm install -g n

npm cache clean -f

sudo n lts
```

npm을 업데이트하고 위의 명령어를 사용하였지만 node -v를 통해 나오는 버전은 변함이 없었습니다.

분명 다운로드에는 문제가 없었습니다.

```terminal
installed : v20.10.0 to /usr/local/bin/node
active : v18.17.0 at /Users/sid12g/.nvm/versions/node/v18.17.0/bin/node
```

문제는 설치방식에 있었습니다. 과거에는 nvm 방식으로 설치하였지만 이번에는 n으로 업데이트를 하였기 때문입니다.

해결방법은 간단했습니다!

```terminal
ln -sf [installed경로] [active경로]
```

위의 명령어로 symbolic link를 생성합니다.

그 후 node -v 명령어를 통해 Node가 업데이트된 것을 확인할 수 있습니다.
